Задания третьего модуля курса "Автоматизация тестирования с помощью Selenium и Python":
1) Оформление теста на уникальность селектора (из первого модуля) в стиле unittest
2) Оформление теста в стиле PyTest
3) Задание на область видимости фикстур - посчитать, сколько смайликов будет напечатано при выполнении тестового класса
4) Задание на пропуск тестов с помощью маркировки xfail
5) Задание по маркировкам тестов
6) Автотест c авторизацией, проходящий по 8 ссылкам, заполняющий формы и получающий значение из результирующих сообщений для составления кодовой фразы
7) Атотест по проверке наличия кнопки добавления товара в корзину для разных языков интерфейса, запускается с помощью параметра: pytest test_items.py —language=fr (указывается любой язык из списка, например, ru или de)
